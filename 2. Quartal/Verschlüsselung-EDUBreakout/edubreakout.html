RÄTSEL 1 / 4 – Caesar-Verschlüsselung

Der Klassiker aus der Antike

Buchstaben verschieben mit Julius Caesar

Was ist die Caesar-Verschlüsselung?

Julius Caesar (100–44 v. Chr.) schickte Befehle an seine Feldherren – aber damit Feinde die Botschaft nicht lesen konnten, verschlüsselte er jeden Buchstaben. Sein Trick: Er verschob jeden Buchstaben um eine feste Anzahl von Stellen im Alphabet.

Bei Verschiebung +3 wird aus A ein D, aus B ein E, aus Z ein C (nach Z kommt wieder A). Zum Entschlüsseln verschiebt man jeden Buchstaben einfach 3 Stellen zurück.

Beispiel: K im Geheimtext → K ist der 11. Buchstabe → 11 minus 3 = 8 → der 8. Buchstabe = H

// NACHRICHT VON AGENT X // CAESAR-VERSCHLÜSSELUNG, VERSCHIEBUNG +3 //
GHLQ FRGH-EXFKVWDEH LVW: K

RÄTSEL 2 / 4 – Atbash-Verschlüsselung

Der Spiegelcode

Das gespiegelte Alphabet

Was ist Atbash?

Atbash stammt aus dem alten Hebräischen und ist eine der ältesten Verschlüsselungen der Welt. Das Prinzip: Das Alphabet wird gespiegelt. Der erste Buchstabe (A) wird zum letzten (Z), der zweite (B) zum vorletzten (Y), und so weiter.

Das Besondere: Atbash ist selbstinvers – Entschlüsseln und Verschlüsseln sind identisch. Wer einen Atbash-Text nochmals durch Atbash schickt, bekommt den Klartext zurück.

Das letzte Zeichen in der Nachricht ist Z. Was ergibt Z im Spiegelcode?

// NACHRICHT VON AGENT X // ATBASH-SPIEGELCODE //
WVRM XLWV-YFXHSGZYV RHG: Z

RÄTSEL 3 / 4 – Morsecode

Punkte und Striche

Die Geheimsprache des Telegraphen

Was ist der Morsecode?

Samuel Morse erfand 1836 ein Codesystem für den Telegraphen: Statt Buchstaben werden kurze Impulse (·) und lange Impulse (–) übertragen. Jeder Buchstabe des Alphabets hat eine eindeutige Kombination aus Punkten und Strichen.

Morsecode wurde bis ins 20. Jahrhundert auf Schiffen und in der Luftfahrt eingesetzt. Der internationale Notruf SOS lautet: ··· – – – ···

Suche in der Tabelle die Zeichenfolge -.-. und finde den zugehörigen Buchstaben.

// NACHRICHT VON AGENT X // MORSECODE //
-.-.

RÄTSEL 4 / 4 – Binärcode

Die Sprache der Computer

Nur zwei Zustände: 0 und 1

Was ist Binärcode?

Computer kennen intern nur zwei Zustände: Strom an (1) oder Strom aus (0). Jede Zahl, jeder Buchstabe, jedes Bild wird letztlich als Folge von Einsen und Nullen gespeichert – das nennt sich Binärsystem.

Jede Stelle hat einen festen Stellenwert (von rechts: 1, 2, 4, 8, 16 ...). Um eine Binärzahl in eine Dezimalzahl umzurechnen, multipliziert man jede Stelle mit ihrem Stellenwert und addiert die Ergebnisse.

Das Ergebnis ist eine Zahl → suche den Buchstaben an dieser Position im Alphabet (A=1, B=2, ...).

// NACHRICHT VON AGENT X // BINÄRCODE → ALPHABET-POSITION //
1011
